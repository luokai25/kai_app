# KAI

An AI built from Luo Kai's own messages — his reflection and companion.

- **Fused UI:** chat in the center, full people-graph in the left drawer, each chat's own memory-vault on the right.
- **Local-first:** the entire resolved corpus (527,936 messages across WhatsApp, Instagram, Snapchat) ships inside the app and works offline.
- **Optional API:** paste any key (Anthropic / Groq / Mistral / OpenAI) — KAI auto-detects the provider. The API sharpens wording; **memory always stays local**.
- **People are resolved across platforms** — Rosé, Rawan, Ruby and others are one identity, not platform-fragments.

Built via GitHub CI (Cordova + Gradle). Sensitive content is preserved in the data but kept out of the everyday voice.
